# SQL Basics

## Common SQL Statements
- `SELECT` → Retrieve data
- `INSERT` → Add data
- `UPDATE` → Modify data
- `DELETE` → Remove data

## Key Concepts
- Tables, rows, columns
- Primary and foreign keys
- Functions (COUNT, AVG, CONCAT, etc.)

## Databases
- MySQL / MariaDB
- Microsoft SQL Server
